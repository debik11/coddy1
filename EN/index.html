Minecraft Teacher

Who do we see in you:

  1. A person familiar with the popular game Minecraft and versed in its basic aspects and mechanics.
  2. Knowledgeable in one or more of the areas below:
    • Programming in Scratch;
    • Lua-Programming;
    • Work with block diagrams, block programming.
    • Functional programming in Python
  3. With experience in working with children from 6 to 10 years.

A little bit about work:

You can choose one of the following directions for working and learning the Minecraft platform:
  • Minecraft in Scratch - programming using the Scratch interface.
  • Minecraft: An Introduction to Artificial Intelligence - long course for children from 7 to 12 years, designed for learning the basics of programming, as well as raising the level soft skills of children.
  • Minecraft: Python programming - a course designed for learning Python through Minecraft with special plugin and library allowing interact with the elements of the world through code.
  • Minecraft Lua Programming - Learning Lua through writing scripts for Minecraft.
